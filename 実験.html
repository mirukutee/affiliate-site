コインチェックとは？

※本記事には広告（アフィリエイトリンク等）が含まれています。

最短で取引開始／全通貨500円から購入／初心者でもスマホで完結

概要

東証プライム上場マネックスグループのグループ会社が運営する暗号資産取引所。
アプリDL数6年連続No.1（2019〜2024年、データ協力：AppTweak）。金融庁登録済の暗号資産交換業者です。
ビットコイン現物取引高でも2ヶ月連続「国内No.1」を獲得（2022年7・8月／自社調べ）。

特徴・サービス

  • 取り扱い通貨数 国内最大級：全通貨500円から購入可能
    ※ 2022年5月 金融庁暗号資産交換業者登録対象／自社調べ
  • 無料のBTC取引手数料：PC版取引所の板取引が対象
  • 貸仮想通貨：一定期間の貸付で所定の利用料を付与
  • 電気・ガスで貯める：「Coincheckでんき／Coincheckガス」で公共料金支払いでBTC付与
  • ポイント交換：マネックス証券のマネックスポイント→BTC/XRP/ETH
  • セキュリティ：ネットワーク分離・全通貨コールドウォレット対応済み
  • 入金手段：銀行振込・コンビニ・クイック（Pay-easy ※原則24時間365日）

ビットコイン積立（毎日／月イチ）

  • プラン：毎日つみたて／月イチつみたて
  • 最低金額：月10,000円（毎日は1日約300円目安）
  • 引落し〜買付まで自動化で手間いらず
  • 時間分散（ドルコスト）で価格変動リスクを平準化しやすい
補足：ドルコスト平均法＝変動する資産を一定額で定期購入し、平均購入単価の平準化を狙う手法。

取り扱い通貨（例）

国内最大級の31種類以上に対応（例）：

  • BTC / ETH / ETC / LSK / XRP / XEM / LTC / BCH / MONA / XLM / QTUM / BAT / IOST / ENJ / SAND / DOT / FNCT / CHZ / LINK / MKR / MATIC / APE / AXS / IMX / WBTC / SHIB / AVAX / BRIL / BC / DOGE / GRT / MANA / MASK / PEPE / DAI

※ 取り扱いは変動します。最新は公式を確認。※ 2022年9月 調べ

登録方法（4ステップ）

  1. 公式サイトへアクセス
  2. メール登録→パスワード設定
  3. 本人確認書類をアップロード（スマホで完結）
  4. 審査完了→入金して取引開始

まずは公式でチェック

初心者でも使いやすいUIと、積立・貸仮想通貨などの便利機能が充実。

注意点・リスク

  • 暗号資産は価格変動が大きく、元本割れの可能性があります。余剰資金での利用を推奨。
  • 取引所のメンテ時間・スプレッド等の条件を必ず確認。
  • アカウントは二段階認証（2FA）を有効化し、公式アプリを最新に保つこと。
